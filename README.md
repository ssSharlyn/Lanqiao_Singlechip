蓝桥杯单片机备赛全攻略：基于调度器的模块化模板本项目是一个专门为 **蓝桥杯全国软件和信息技术专业人才大赛（单片机设计与开发组）** 打造的开源代码库。仓库包含了高度封装的底层驱动、基于时间片轮询的调度器模板，以及由作者Sharlyn基于该模板编写的历届省赛、国赛高分参考代码。

🌟 项目亮点

- 🚀 核心源自名师：本项目所有底层驱动及调度器架构均基于 Bilibili 知名 Up 主 Alice_西风 的教程实现。代码逻辑严谨，非常适合正在学习西风老师教程的同学直接上手实战。
- ⏳ 非阻塞式调度器：采用西风老师推荐的时间片轮询架构，彻底告别阻塞式的 delay 延时，显著提升多任务（如数码管扫描与传感器读取）的并发处理能力。
- 📦 全模块驱动支持：完整包含 LED、数码管、独立/矩阵按键、DS1302、DS18B20、PCF8591、AT24C02、超声波及频率测量等外设的底层代码。
- 🏆 满分级实战：收录近年省赛、国赛完整解决方案，所有代码均经过真机测试，逻辑清晰，符合竞赛评卷高分标准。

🚀 核心架构

说明本仓库代码严格遵循西风老师的开发思维：底层驱动层：对 onewire、iic 等底层协议进行二次封装，提供极其简洁的 API 接口。调度器层：利用定时器产生 1ms 中断，在中断中维护各任务执行标志位，主循环根据标志位分时执行任务。应用逻辑层：逻辑编写完全解耦，开发者只需在 main.c 中调用封装好的驱动函数即可完成复杂功能。
