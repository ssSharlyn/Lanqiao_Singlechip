本项目是一个专门为**蓝桥杯全国软件和信息技术专业人才大赛（单片机设计与开发组）**打造的开源代码库。仓库包含了高度封装的底层驱动、基于时间片轮询的调度器模板，以及由本人编写的历届省赛、国赛的高分参考代码。

🌟 项目亮点
非阻塞式调度器：摒弃传统的 delay 延时函数，采用基于定时器中断的时间片轮询调度架构，显著提高代码执行效率与响应速度。

模块化驱动：包含 LED、数码管、独立/矩阵按键、DS1302、DS18B20、PCF8591、AT24C02、超声波及频率测量等所有核心外设的底层代码。

实战导向：收录了近年省赛、国赛的完整解决方案，所有代码均经过真机测试，符合满分要求。

教学兼容：本仓库的代码逻辑与 Bilibili 知名 Up 主 Alice_西风 的教程体系高度兼容，方便学习者快速上手。

架构模式：时间片轮询调度 (Task Scheduler) + 状态机按键识别

🚀 核心架构说明
本模板的核心在于调度器文件。其设计逻辑如下：

底层驱动层：对 onewire、iic 等底层协议进行二次封装，提供简洁的 API 接口。

调度器层：利用定时器产生 1ms 中断，维护各任务的执行频率（如数码管 2ms 刷新，按键 10ms 扫描，传感器 500ms 采样）。

应用逻辑层：在 main.c 中仅需关注业务逻辑的编写，无需考虑底层的相互干扰。
